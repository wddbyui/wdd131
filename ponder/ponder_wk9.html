Ponder - Flexbox

  • Flexbox

    Flexbox is a layout method for arranging items in rows or columns.

    It is different from grid in that flexbox is one-dimensional where grid is two-dimensional. With grid you have control over both rows and columns and flexbox is along a single axis.

    Grid focuses on overall page structure and items can be placed anywhere in the grid, whereas flexbox focuses on flow and positioning and items flow naturally in order.

                        
.main-nav {				
    display: flex;					
    justify-content: flex-end;
    gap: 20px 
}
                    
                  
                        
main {
    display: flex;
    flex-wrap: wrap;
    gap: 50px;
}
