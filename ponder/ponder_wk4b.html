Ponder - Responsive Gallery with Modals

  • dialog

    The HTML dialog tag defines a dialog box or subwindow

    Used to create modals or popup dialogs.

    A dialog is hidden by default

    Use .close(); and .open(); to open and close a dialog/modal in JS.

  • Event parameter

    When you use the .addEventListener() method in JS, the function you call to handle the event can have a special parameter which is typically called 'event' or 'e'.

    This parameter is an object that contains various information and methods related to the specific event that occured.

    You can use this event parameter to give you control over how the event behaves.

    It does not have to be explicitly sent as an argument in the call to the function.

                            
gallery.addEventListener('click', showModal);

function showModal(e) {
    console.log(e);
    console.log(e.target);
}

  • .replace()

    The .replace() method in JS searches a string for a value and returns a new string.

    It does not change the original string.

                            
const paragraph = "I think Max's code is more efficient than yours.";

console.log(paragraph.replace("Max's", "my"));
// Expected output: "I think my code is more efficient than yours."

  • position

    The position CSS property is used to specify the type of positioning method used for elements.

    'static' is the default and are positioned in the normal HTML flow of the page.

    'absolute' is positioned relative to the positioned parent container

    'fixed' is positioned relative to the viewport, staying in place as the user scrolls.

    'relative' is positioned relative to its normal position in HTML flow.

    'sticky' is positioned based on the user's scroll position. Like relative and fixed. Relative until a given offset is met and then it sticks in place like fixed.

    You can specify 'top', 'right', bottom, or left measurements to position the element.

                            
.close-viewer {
    position: absolute;
    top: 10px;
    right: 20px;
}

Modals

Each image in this activity has a lower and higher resolution version. The lower resolution images are used in the gallery thumbnails and the higher resolution image is used in the modal.

Download the gallery images and add them to your project folder inside an images folder.

When the user clicks the thumbnail the same higher resolution image shows in the modal.

The modal can be closed with the close button with the 'X' by clicking outside the modal image, or by pressing the 'Esc' key.
