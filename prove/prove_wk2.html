Computational Thinking

Using our ponder theme changer activity as a guide, think through the steps that will be needed to create this project.

We have not gone over event listeners, functions, and conditional if clauses, but using the ponder activity as a guide will help.

  • HTML addition:

                        
<select>
    
    <option>Choose Mode</option>
    
    <option value="dark">Dark</option>
    
    <option value="light">Light</option>
    
</select>

    Beginning JavaScript:

                        
let selectElem = document.querySelector('select');
let logo = document.querySelector('img');

selectElem.addEventListener('change', changeTheme);

function changeTheme() {
    let current = selectElem.value;
    if (current == 'dark') {
        // code for changes to colors and logo
    } else {
        // code for changes to colors and logo
    }
}

What should be familiar to you is the way you can use the DOM to change the look of the page. For example, with this assignment, it would be the code within the if and else clauses.
