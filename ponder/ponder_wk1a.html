Ponder - Meta Tags & CSS Review

  • meta tag

    The meta tag defines metadata or information about your HTML document.

    meta tags always go inside the head element and are used to specify the character set, the page description, the author of the document, viewport settings, etc.

    The browser will use this data but they do not display on the page.

    •                         
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="An in-depth guide to HTML meta tags.">
<meta name="author" content="Jane Doe">

  • CSS Selectors

    Selector Example Description
    element h2 Selects all h2 elements
    #id #topics Selects the elements with id="topics"
    .class .list Selects all elements with the class="list"

  • CSS Combinators

    A combinator is a sign that explains the relationship between selectors.

    Sign Example Result
    (a single space) div p Selects the p elements inside the div
    , div, p Selects all div elements and all the p elements
    > div > p Selects all p elements that are direct children of a div
    + div + p Selects the first p elements that is immediately after the div
    ~ h2 ~ p Selects all the p elements that are preceded by a h2 element

  • Pseudo-Class Selectors

    Example Description
    a:hover Selects the a element that is being hovered over
    li:nth-of-type(2) Selects only the second li
