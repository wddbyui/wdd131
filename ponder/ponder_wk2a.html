Ponder - DOM Manipulation

  • .querySelector()

    The .querySelector() method returns the first child element that matches a specified CSS selector of an element.

    •                         
// Targeting an element in the HTML with a class of 'example' and giving it a new background color.                         
document.querySelector(".example").style.backgroundColor = "red";

  • .getElementById()

    The .getElementById() method returns an element with a specified id.

    •                         
// Targeting an element in HTML with the id of content and assigning it to a constant.
const myElement = document.getElementById("content");

  • .setAttribute()

    The .setAttribute() method sets a new value to an attribute.

    It contains two required parameters. The name of the attribute and the new value.

    •                         
// Setting the image element in HTML with a new source attribute.
document.getElementById("myImg").setAttribute("src", "portrait.webp");

  • .textContent()

    The .textContent property sets or returns text content.

    •                         
// Setting an elements HTML text content.
document.querySelector("elem").textContent = "I have changed!";

  • .style

    The .style property can returns or assign the values of an elements style attribute.

                            
// Setting the font color of an element to red                          
document.getElementById("myH1").style.color = "red";

  • .className

    The .className property sets or returns an element's class attribute.

                            
// Assigning a new class of mystyle to an element
document.getElementById("myDIV").className = "mystyle";

  • .classList

    The .classList property returns the CSS classnames of an element.

                            
// Adding the 'myStyle' class to an element.
const list = element.classList;
list.add("myStyle");

    .classList can also be used with .remove() and .toggle()

  • .value

    The .value property sets or returns the value of an attribute.

    Commonly used to get user input from input boxes or dropdown menus.

                            
// Getting the current value from an element.
let elem = document.querySelector('select');
let current = elem.value;

DOM stands for Document Object Model which is refers to the HTML elements on a web page. 'DOM Manipulation' is using JavaScript to make changes after the page has loaded. This includes changing the structure, style, or content of a web page by adding, removing, or updating HTML elements or attributes.
