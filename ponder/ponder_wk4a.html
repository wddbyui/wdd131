Ponder - Responsive Menu

  • .classList

    The classList property returns the CSS classnames of an element.

    Adding a class called 'hide'

                            
element.classList.add('hide');

    Removing a class called 'hide'

                            
element.classList.remove('hide');

    Toggling a class called 'hide' on and off

                            
element.classList.toggle('hide');

  • transform

    The transform property allows you to rotate, scale, move, skew, etc., elements.

    translate

    Used with transform to move an element.

    rotate

    Used with transform to rotate an element.

    This example moves the bars along the y axis and rotates it 45 degrees.

                            
.change .bar1 {
  transform: translate(0, 11px) rotate(-45deg);
}
