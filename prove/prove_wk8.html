Instructions:

Add social media meta information to the head of the HTML.

  • Social Media Meta tags

    These meta tags give social media sites the information they need to display a nice summary of our site for easu posting to social media.

    Here's an example of recommended tags (change the values to reflect the content of your page):

                            
<meta property="og:title" content="Page Title">
<meta property="og:type" content="website" />
<meta property="og:description" content="Page Description">
<meta property="og:image" content="URL to an image">
<meta property="og:url" content="URL of the page">
<meta name="twitter:card" content="summary_large_image">

    og stands for Open Graph, a format developed by Facebook that many social media sites understand now. There are a lot of options for properties to include. The list above is considered a good starting point for most sites.

Create a character card with: properties: name, class, level, health, and image and methods: attacked and levelUp

The .attacked() method will subtract 20 from the health property value. When the health property reaches zero, let the user know the character has died.

The .levelUp() method will add 1 to the level property value.

This demonstrates how it will run.
